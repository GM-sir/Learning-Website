HTML Text Formatting Tags

HTML text formatting tags are used to style and structure text content in web pages. These tags add meaning or presentation styles like bold, italic, underlined, or highlighted text.

Common Text Formatting Tags 

 
   
This is bold text.

   
This is important text. 

   
This is italicized text. 

   
This is emphasized text. 

   
This is underlined text. 

   
This is highlighted text. 

   
This is smaller text. 

   
This is deleted text. 

   
This is inserted text. 

   
H2O is the chemical formula for water. 

   
E=mc2 is Einstein's equation.

Example of All Text Formatting Tags Combined 

    <!DOCTYPE html>
    <html> 
    <head>
      <title>Text Formatting Example</title>
    </head>
    <body> 
      <p>This is <b>bold</b>, <strong>strong</strong>, <i>italic</i>, and <em>emphasized</em> text.</p>
      <p>This is <u>underlined</u>, <mark>highlighted</mark>, and <small>smaller</small> text.</p>
      <p>This is <del>deleted</del>, <ins>inserted</ins>, <sub>subscript</sub>, <sup>superscript</sup> text.</p>
    </body>
    </html>
This example demonstrates the various formatting tags in a single document. When rendered in a browser, each tag will apply the respective style.
