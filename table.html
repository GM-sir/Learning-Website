Tables

Basic Table Tags
  • <table>: Defines the table.

  • <tr>: Represents a table row.

  • <th>: Represents a table header.

  • <td>: Represents a table cell.

Code Example:

    <table border="1"> 
      <tr>
        <th>Name</th> 
        <th>Age</th> 
        <th>City</th>
      </tr> 
      <tr> 
        <td>John</td> 
        <td>25</td> 
        <td>New York</td>         
      </tr>
      <tr> 
        <td>Jane</td>
        <td>30</td>
        <td>London</td>
      </tr> 
    </table>

     <!DOCTYPE html> 
     <html lang="en">
     <head> 
       <meta charset="UTF-8"> 
       <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
       <title>Table with Colspan and Rowspan </title>      
     </head> 
     <body> 
       <table border="1"> 
     <!-- Table Header --> 
         <tr> 
           <th>Category </th>
           <th>Item </th> 
           <th>Quantity </th>          
         </tr> 
     <!-- Row 1 --> 
         <tr> 
           <td rowspan="2">Fruits </td>   <!-- Spans 2 rows -->
           <td>Apple </td> 
           <td>10 </td>           
         </tr>
    <!-- Row 2 -->
         <tr> 
           <td>Banana </td>
           <td>20 </td>         
         </tr> 
    			  	<!-- Row 3 --> 
          <tr> 
            <td>Vegetables</td> 
            <td colspan="2">Carrot and Spinach             
          </tr>       
       </table>  
     </body> 
     </html>

Explanation

  1. rowspan:
    • Used in the "Fruits" cell to merge it across two rows.

    • Value rowspan="2" indicates the cell spans 2 rows.

  2. colspan:
    • Used in the "Carrot and Spinach" cell to merge it across two columns.

    • Value colspan="2" indicates the cell spans 2 columns.

Output


Table with Colspan and Rowspan
Category Item Quantity
Fruits Apple 10
Banana 20
Vegetables Carrot and Spinach

This creates a neat table with merged cells without using CSS.
