Links and Images

Anchor Tag ( <a>)

The <a> tag is used to create hyperlinks. It requires the href attribute to specify the link's destination.

Code Example:

<a href="https://www.example.com" title="Visit Example">Visit Example </a>
  • href: Specifies the URL.

  • title: Displays a tooltip when hovered over.

Image Tag ( <img>)

The <img> tag is used to embed images into a webpage. It requires src (source URL) and alt (alternative text) attributes.

Code Example:

<img src="https://www.example.com/image.jpg" alt="Sample Image" title="Image Title">
  • src: Path to the image.

  • alt: Text displayed if the image fails to load.

  • title: Tooltip text for the image.
