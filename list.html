HTML List Tags

HTML provides several types of list tags to display items in an organized manner. There are three main types of lists in HTML:

1. Ordered List (<ol>)

An ordered list displays items in a numbered or sequential order.

Syntax: 

     <ol>
       <li> Item 1 </li> 
       <li> Item 2 </li> 
       <li> Item 3 </li> 
     </ol>
Output:
  1. Item 1

  2. Item 2

  3. Item 3


2. Unordered List (<ul>)

An unordered list displays items with bullet points, regardless of order. Syntax 
     <ul> 
       <li> Item A</li> 
       <li> Item B</li>  
       <li> Item C</li> 
     </ul>
Output:
  • Item A

  • Item B

  • Item C


    • 3. Definition List ( <dl> )

    A definition list is used to display terms and their descriptions. Syntax: 
        <dl>
      <dt>HTML</dt> 
      <dd>A markup language for creating web pages.</dd>
      <dt>CSS</dt>
      <dd>A style sheet language for designing web pages.</dd>           
    </dl>
    Output:

    HTML

    A markup language for creating web pages.

    CSS

    A style sheet language for designing web pages.

    Attributes in List Tags


    1. Type Attribute for <ol>

    Specifies the type of numbering (e.g., numbers, Roman numerals, letters). 
        <ol type="I"> 
      <li>Item 1</li>
      <li>Item 2</li> 
    </ol>
    Output:
    1. Item 1
    2. Item 2

    2. Type Attribute for <ul>

    Changes the bullet style (e.g., circle, square, disc). 
        <ul style="list-style-type: square;"> 
      <li>Item A</li> 
      <li>Item B</li> 
    </ul>
    Output:
    • Item A
    • Item B

    Nesting Lists

    Lists can be nested to create complex structures.

    Example:

        <ul> 
      <li>Fruits 
        <ul> 
          <li>Apple</li> 
          <li>Banana</li>              
        </ul> 
      </li> 
      <li>Vegetables 
        <ol> 
          <li>Carrot</li> 
          <li>Broccoli</li>              
        </ol> 
      </li> 
    </ul>
    Output:
    • Fruits
      • Apple
      • Banana
    • Vegetables
      1. Carrot
      2. Broccoli

    Example of All List Types Together 

        <!DOCTYPE html> 
    <html> 
    <head> 
      <title>HTML Lists Example</title>           
    </head>
    <body>
      <h2>Ordered List</h2> 
      <ol> 
        <li>Step 1</li> 
        <li>Step 2</li>
        <li>Step 3</li>             
      </ol>
          
      <h2>Unordered List</h2> 
      <ul> 
        <li>Option A</li>
        <li>Option B</li> 
        <li>Option C</li>           
      </ul>
          
      <h2>Definition List</h2> 
      <dl> 
        <dt>JavaScript</dt>
        <dd>A scripting language for web development.</dd>
        <dt>Python</dt> 
        <dd>A programming language for various applications.</dd>            
      </dl> 
    </body>
    </html>
    When rendered in a browser, this displays all three types of lists with their distinct formatting.
