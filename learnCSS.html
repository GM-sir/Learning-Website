1. What is CSS?

CSS (Cascading Style Sheets) is a style sheet language used to describe the look and formatting of a document written in HTML. It is used to enhance the presentation of web pages by defining styles such as layout, colors, fonts, and spacing.

Types of CSS:

  1. Inline CSS: Defined directly in an HTML tag using the style attribute.

  2. Internal CSS: Written inside a <style> tag within the <head> section of the HTML file.

  3. External CSS: Placed in an external file (e.g., style.css) and linked using a <link> tag.

2. CSS Syntax

CSS syntax consists of selectors, properties, and values. 
    /* Example Syntax */ 
  selector 
  { 
    property: value; 
  }

  • Selector: Selects the HTML element to style.

  • Property: A style attribute like color, font-size, or margin.

  • Value: Specifies the propertyâ€™s style.

Example: 
    p { 
      color: blue; 
      font-size: 16px;
    }

3. Basic Selectors

a. Universal Selector (*)
Applies to all elements. 
    * {
      margin: 0; 
      padding: 0;
    }
b. Type Selector
Targets specific HTML tags like <h1>, <p>, etc. 
    h1 {
      color: green; 
    }

c. Class Selector (.classname)

Styles elements with a specific class attribute. 
    <p class="highlight">This is a highlighted paragraph.</p>
    .highlight { 
      background-color: yellow; 
    }

d. ID Selector (#idname)

Styles a unique element with an id attribute. 
     <h1 id="main-title">Welcome to CSS </h1>
    #main-title { 
      font-size: 24px; 
      text-align: center;
    }

4. Adding CSS to HTML

a. Inline CSS
Styles are applied directly to the HTML element using the style attribute. 
    <p style="color: red; font-size: 20px;">This is an inline-styled paragraph.</p>
b. Internal CSS
Styles are written inside a <style> tag within the <head> section. 
      <!DOCTYPE html> 
      <html lang="en"> 
     <head> 
        <style> 
        p { 
          color: blue;
          font-size: 18px;
        }
        </style> 
      </head> 
      <body> 
        <p>This is an internally styled paragraph.</p>
      </body> 
      </html>
